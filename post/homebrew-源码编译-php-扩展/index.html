HomeBrew 源码编译 PHP 扩展

Fri, Apr 28, 2017

如果你需要装个 mongodb 扩展，只需要执行下列代码就可以进行编译安装，HomeBrew已经自动为你添加 php.ini 配置内容，非常快捷！

brew reinstall php70-mongodb --build-from-source
